## 🚀 How to Run

Clone the repository

```bash
git clone https://github.com/p1nak/ML-WORK.git
```

Navigate into the project

```bash
cd ML-WORK
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```
